# 文档门户

这是一份更适合从 GitHub 首页直接点开的入口页。

## 最快路径

如果你只想快速看懂这个仓库，按这个顺序：

1. `README.md`
2. `docs/PORTAL.md`
3. `docs/REPO_GUIDE.md`
4. `docs/STRUCTURE.md`
5. `docs/INDEX.md`
6. `main.py`
7. `src/config.py`
8. `src/core/pipeline.py`
9. `src/search_service.py`

## 你可能在找什么

### 想知道仓库怎么分块
- `REPO_GUIDE.md`

### 想知道哪些是核心、哪些是运行产物
- `STRUCTURE.md`

### 想知道 docs 目录里每份文档怎么分组
- `INDEX.md`

### 想找部署、FAQ、完整说明
- `INDEX.md`
- `full-guide.md`
- `FAQ.md`
- `DEPLOY.md`

## 轻量版理解

- `README.md`：首页
- `PORTAL.md`：文档门口
- `REPO_GUIDE.md`：仓库分块
- `STRUCTURE.md`：结构图
- `INDEX.md`：文档分组

## 说明

这几个页面不是重复，是分层：
- 首页给外部看
- 门户给第一次进入的人看
- 导览给想理解结构的人看
- 索引给想找具体文档的人看
